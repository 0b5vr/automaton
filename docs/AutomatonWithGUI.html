Constructor

Parameters

_props.loop:boolean (optional)

Whether let the time loop or not

_props.fps:number (optional)

If this is set, the clock will become frame mode

_props.realtime:boolean (optional)

If this is true, the clock will become realtime mode

_props.gui:DOM (optional)

DOM element where you want to attach the Automaton GUI

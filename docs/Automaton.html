Automaton

IT'S AUTOMATON!
It's automaton.nogui.js version and also base class for AutomatonWithGUI.

Constructor

new Automaton(_props)

Parameters
_props:object
_props.loop:boolean (optional)

Whether let the time loop or not

_props.fps:number (optional)

If this is set, the clock will become frame mode

_props.realtime:boolean (optional)

If this is true, the clock will become realtime mode

_props.onSeek:function (optional)

Will call when the method seek() is called

_props.onPlay:function (optional)

Will call when the method play() is called

_props.onPause:function (optional)

Will call when the method pause() is called

_props.data:string

Data of the automaton in JSON format. Required in noGUI mode

Automaton Playground!

Hi there! This is Automaton playground. You can code anything (not dangerous) on editor and tweak parameters on Automaton GUI. GitHub (Download): https://github.com/fms-cat/automaton

What is Automaton?

Automaton is animation engine with Timeline GUI for creative coding. I hope it helps your creative coding! Automaton is still under development so some features will be added / changed in future. First you might want to define a function named auto like var auto = automaton.auto . auto is magical function, it pours cool animation into your code! When you use auto like auto( "rectX" ) for the first time, new parameter channel named "rectX" will be created on Automaton, and it appears on GUI. Then, you can tweak the channel using timeline! Be careful that you cannot use these Automaton channels outside of your update loop. Doing that will create your channel on Automaton, but not updated. Also, don't forget to call automaton.update() !

How to use GUI

The Automaton GUI has a lot of feature! First, choose the channel you want to tweak by left menu. Then the GUI will be show animation curve on the timeline. When you doubleclick inside the timeline, new node will be created. The animation curve will transform to pass through nodes. You can grab and move around the node by dragging. And you doubleclick the node, the node will be removed. You also can put cutting-edge Fxs on this timeline. Right-click on the timeline will show you a context menu, then clicking Add Fx will show you a list of Fx definition. Fxs deforms your animation curve into various form. Try various! Or do you still want to do something wacky? Call automaton.addFxDefinitions() to define your own Fx definitions! You can scroll around the timeline using mouse wheel. Shift-Scroll lets the timeline horizontal zoom. Ctrl-Scroll lets the timeline vertical zoom.

Saving GUI changes

Once you tweaked animations using GUI, you might want to save your changes. Saving can be performed by clicking floppy icon at rightmost of GUI. It will send very long JSON string to your clipboard. You can load this JSON savedata by adding a parameter named data when you initialize Automaton.
